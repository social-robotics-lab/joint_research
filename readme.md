# はじめに
ELANから出力された2者会話の書き起こしを分析するための簡単なプログラムのサンプル群です。

### sample.txt
ELANから出力された書き起こしファイルのサンプルです。

### practice.py
サンプルファイルを読み込む練習用のプログラムです。

### practice2.py
サンプルファイルを読み込み、同じ内容をファイルに出力するプログラムです。

### split_morpheme_1.py ~ split_morpheme_4.py
サンプルファイルを読み込み、形態素解析して、ファイル出力するプログラムです。
それぞれのプログラムの中に説明が書かれています。
